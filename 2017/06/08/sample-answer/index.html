1) the code is well written
2) You have a couple of variables named “Key”. Never start a variable name with a capital letter.
3) The breakup function is almost correct. You need to be carful about truthiness - in this case it is a problem with “result[key] ? …”. Try result.hasOwnProperty(key) ?
